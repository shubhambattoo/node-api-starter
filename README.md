# Node API Starter

## Quick Start

Clone this Repository and Install all the dependencies
```
yarn
```

or if you are using NPM

```
npm i
```

after that to serve the project run

```
yarn start
```

or if you are using NPM
```
npm start
```

this will serve the project on port 3000
